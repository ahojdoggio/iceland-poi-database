## Data Notes

### Extraction Method
- Strings extracted from Garmin `.LBL` files using Latin-1 decoding
- Filtered for tourist-relevant POI patterns
- Cross-referenced against existing database to identify missing entries
- Categorized by keyword matching

### Data Quality
- Names are from official Icelandic cartographic sources
- Some entries may include metadata fragments (road numbers, hours)
- Icelandic special characters preserved (á, é, í, ó, ú, ý, þ, ð, æ, ö)
- Duplicate checking performed against current 479 POI database

### Required Enrichment
For each POI to be added to database:
1. GPS coordinates (required - not in source data)
2. Address with postal code
3. Description (1400-1800 characters)
4. Category assignment
5. Website/phone (if available)
6. Opening hours (if applicable)
7. Images from copyright-free sources
